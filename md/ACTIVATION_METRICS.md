## 3. COMMON DROP-OFF POINTS

| Stage | Problem | Fix |
|-------|---------|-----|
| Visitor → Signup | Confusing landing page | Improve value prop |
| Signup → Wizard | Dashboard overwhelm | Clear CTA "Mulai Setup" |
| Wizard → Completed | Step 2 too hard | Pre-fill, make optional |
| Completed → Upload | No documents ready | Provide samples, accept URL |
| Upload → Publish | Don't understand embed | 1-click copy, hosted URL |
| Publish → Conversation | No visitors | Share widget, test yourself |
| Conversation → Lead | Bot doesn't ask | More natural lead capture |
| Lead → Paid | Price too high | Lower price, show ROI |
